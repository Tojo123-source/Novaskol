# Lire avant installation de Novaskol

Bienvenue dans Novaskol.

Ce guide est ecrit pour une ecole qui veut installer Novaskol facilement, sans toucher au code.

## Installation simple sur Windows

1. Double-cliquez sur :

```text
Novaskol-Setup-x64.exe
```

2. Suivez les etapes de l'installateur.
3. A la fin, ouvrez l'application **Novaskol** depuis le Bureau ou le menu Demarrer.
4. Attendez le lancement complet.
5. Si la page d'installation s'affiche, remplissez :
   - le nom de l'ecole ;
   - l'annee scolaire ;
   - les informations de contact ;
   - le premier compte administrateur.

## Choix important au premier demarrage

Pendant l'installation initiale de l'ecole, Novaskol demande de choisir :

- `Base vide` : pour une vraie ecole qui commence sa gestion.
- `Mode demo` : pour tester avec des donnees fictives.

Si vous commencez avec le mode demo, vous pourrez revenir au mode reel plus tard depuis l'installation, avec un compte administrateur.

## Utilisation quotidienne

- Ouvrez simplement l'application **Novaskol**.
- Connectez-vous avec votre compte.
- Fermez l'application normalement quand vous avez termine.

## Sauvegardes

Faites une sauvegarde regulierement depuis le module :

```text
Parametre > Sauvegardes
```

Gardez les fichiers de sauvegarde dans un endroit sur :
- cle USB ;
- disque externe ;
- cloud de l'ecole.

## Diagnostic

Apres installation, connectez-vous en administrateur et ouvrez :

```text
Parametre > Diagnostic systeme
```

Si tout est marque `Pret`, Novaskol est correctement prepare.

## Utilisation sur plusieurs appareils dans la meme ecole

Si l'ecole veut utiliser Novaskol sur un PC de bureau, un portable ou un telephone en meme temps :

1. Choisissez un seul **appareil principal** dans l'ecole.
2. Lancez Novaskol sur cet appareil principal.
3. Ouvrez dans Novaskol :

```text
Parametre > Reseau local
```

4. Recuperez l'adresse locale de l'ecole ou le QR code.
5. Connectez les autres appareils au **meme Wi-Fi** ou au meme routeur local.
6. Ouvrez cette adresse locale depuis l'autre appareil.

Important :

- l'appareil principal doit rester allume ;
- internet n'est pas obligatoire ;
- tous les appareils doivent etre sur le meme reseau local.

## Important

- Ne partagez jamais la meme base de donnees entre deux ecoles.
- Gardez le mot de passe administrateur dans un endroit sur.
- Faites une sauvegarde avant toute grosse modification.
- Si l'ecole veut utiliser Novaskol en ligne, utilisez le guide `docs/HEBERGEMENT.md`.
